The requirements for this model is -

We must have installed SimulIDE & Atmel Studio for working on this project.

1. .hex file of model.
2. Voltimeters.
3. Led.
4. atmega328P.
5. Voltage source.
6. Resistor(100ohms).
7. Switch.
8. Ground Sources.
